# Karman (working title)

A native macOS fluid-dynamics instrument: real-time GPU lattice-Boltzmann CFD
where **every run ships an honest error bar** — verification, numerical
uncertainty, statistical confidence, and regime validity, built into the
instrument instead of left as an exercise for the user.

Status: planning. See [PLAN.md](PLAN.md) and [docs/research/](docs/research/).
